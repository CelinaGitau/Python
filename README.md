📌 Project Overview

This project explores the relationship between income levels and happiness scores. The goal was to determine whether a correlation exists and to transform raw data into meaningful insights using google colab  data analysis and   pandas, matpolit and seaborn and visualisation techniques.


🛠️ Tools & Technologies

Google Colab – for running and documenting the analysis
python pandas – for data cleaning and manipulation
matplotlib – for creating visualisations
seaborn – for enhanced and styled data visualisation


📈 Methodology

The dataset was first cleaned and organised using pandas.
A scatter plot was selected as the main visualisation because it effectively displays  correlation relationships between two variables.

Why this is important:

- Clear identification of trends
- Easy detection of patterns
- Visual assessment of correlation (positive or negative)


🔍 Key Insights

- The visualisation highlights how income levels relate to happiness scores
- Trends in the scatter plot help determine the strength and direction of correlation
- Outliers can be easily identified and analysed


✅ Conclusion

This project demonstrates how data visualisation tools can be used to uncover relationships within data. By combining statistical analysis with clear visual outputs, complex datasets can be made easier to understand and interpret.


The scatter plot created using pandas, matplotlib, and seaborn shows a positive correlation between income and happiness levels. This means that, within this dataset, higher income is generally associated with higher levels of happiness.


However, this data visualisation  represents only one aspect of the provided dataset of  reality and should be interpreted with caution. A key limitation is that income does not always determine happiness, other factors such as health , relationships, and lifestyle can also play a significant role.


Overall, the project highlights how effective visualisations make patterns in data easy to identify, while also emphasising the importance of critical thinking when drawing conclusions from data.
